📤 What you send up — Friday 12pm report
Friday 12pm Report to Admin Head — HR Pipeline + Attrition RAG · 5 lines max
Admin Head uses this to build the Headcount section of the Friday 3pm CEO briefing. Late = incomplete CEO briefing.
Open roles
Role · stage (sourcing / shortlisted / interviews / offer) · days open · on track / at risk.
Pipeline this week
Interviews held · decisions pending · offers sent / accepted / declined.
Attrition signal
Any resignation received or at-risk signal observed this week. If none — "No attrition signal."
Compliance status
Statutory filings current · any employee file gap · any POSH flag. If clean — "Compliance clear."
Flags & asks
Anything needing Admin Head decision (e.g. DH not reviewing longlist · role open 3 weeks). If none — "No flags."
Bypass Friday for these — notify Admin Head same hour: Any senior resignation (Practice Head or above) · Any statutory compliance breach discovered · Any candidate harassment complaint received.
Bypass Friday for these — notify Admin Head same day: Role open >3 weeks with no approved shortlist · Candidate accepted offer elsewhere because of slow feedback · Day 1 readiness failure (tool not live).
HR Manager Must NOT Do These
✕
Make technical hire/no-hire decisions — DH decides technical fit. HR decides process fit.
✕
Post roles or source against an unapproved brief — wait for Admin Head approval before any outreach.
✕
Communicate salary offers without Admin Head and CEO approval — pre-approved band only.
✕
Conduct performance management independently — HR supports and documents. Line managers own.
✕
Wait for Friday to tell Admin Head about a resignation — same hour for senior, same day for all.
